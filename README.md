# 文档对比助手 (MatchDiff) 说明文档

## 项目概述
文档对比助手是一款基于原生 Web 技术开发的轻量级前端核对工具。它支持多种办公文档格式的内容解析，通过双视窗同步比对算法，自动识别并高亮文档间的差异点，旨在提升人工校对修订文档的精确度与效率。

## 核心功能需求实现

### 1. 多格式解析与兼容性
* **引擎集成**：采用 `SheetJS` (Excel)、`Mammoth.js` (Word)、`PDF.js` (PDF) 及 `jsdiff` (文本算法) 核心库。
* **Excel 多表处理**：支持自动识别 `.xlsx/.xls` 文件内的所有工作表（Sheets），并提供独立的选择器进行目标切换。
* **本地化处理**：所有文档解析均在客户端浏览器完成，不上传服务器，确保数据隐私安全。

### 2. UI 布局与交互体验
* **紧凑型视窗**：压缩页头区域，最大化对比工作区的纵向空间。
* **高强度视觉引导**：上传区域采用加粗、加大的蓝色文字（16px, Weight 900）引导。
* **动态拖拽反馈**：文件悬停在上传区时，触发边框加粗及背景色深度切换，提供明确的预判反馈。

### 3. 对比引擎规格
* **零抖动同步滚动**：左右视窗通过“逻辑锁”机制同步 `scrollTop` 与 `scrollLeft` 属性，消除循环触发导致的界面闪烁。
* **差异化展示**：
    * **表格模式**：基于坐标对齐，内容冲突单元格应用红色背景及边框高亮。
    * **文本模式**：基于行差异算法，绿色高亮代表新增内容，红色删除线代表已移除内容。
* **结构化标识**：文本模式下自动解析并插入 PDF 原生页码分隔条。

### 4. 高级交互逻辑
* **侧边栏差异清单**：实时汇总所有差异项，支持统计差异总数。
* **精准跳转定位**：点击清单条目，主视图调用 `scrollIntoView` 接口将对应差异点平滑滚动至屏幕**中心**位置，并辅以临时橙色高亮。
* **动态视窗管理**：
    * **宽度调节**：通过中间分割条（Resizer）支持鼠标拖拽，实时调整侧边栏宽度（180px - 500px）。
    * **抽屉式隐藏**：侧边栏支持彻底隐藏以扩大视野；隐藏后通过右下角“悬浮球”重新唤起，实现完整的状态闭环。

## 技术规格与环境
* **开发语言**：HTML5, CSS3, JavaScript (ES6+)。
* **运行环境**：Chrome, Edge, Firefox, Safari 等现代主流浏览器。
* **部署方式**：单 HTML 文件部署，无需配置服务器依赖。

## 注意事项
1.  **解析限制**：PDF 解析仅限于文本层提取，不支持图片扫描件格式。
2.  **比对范围**：当前版本侧重于文本内容比对，暂不包含单元格格式（颜色、字体大小）的样式比对。
3.  **性能参考**：大规模 Excel 数据（数万行以上）的解析速度受限于本地设备硬件性能。
